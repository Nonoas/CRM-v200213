# 软件介绍

本软件是基于 JavaFX8 的一款单机的小型店铺管理系统，数据库使用sqlite3，
无需用户安装数据库软件，也无需软件方提供远程数据库。功能包括 商品管理、用户管理、员工管理、套餐管理。报表统计等功能。
本软件将持续更新维护。
